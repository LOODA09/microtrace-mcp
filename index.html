⚡ MicroTrace

Automated SBOM Extractor for Embedded Binaries

An intelligent system for firmware reverse engineering, component classification, and vulnerability analysis. MicroTrace automatically generates a Software Bill of Materials (SBOM) from bare-metal embedded firmware, identifying software layers and presenting them through an interactive web visualization.

Explore Function Graph
🔍

Automated Disassembly

Process firmware binaries using Ghidra Headless to extract functions, symbols, and call relationships. Export results in structured JSON for comprehensive analysis.

🎯

Component Classification

Apply YARA rules to detect MCAL functions, HAL functions, and application logic. Automatically classify software layers for better understanding of firmware architecture.

📊

Interactive Visualization

Display results on a web-based Node Graph with nodes representing functions or modules and edges showing function calls or dependencies. Color-coded by software layers.

🛡️

Vulnerability Detection

Cross-check identified libraries with CVE databases. Alert users to known vulnerabilities and generate structured SBOM reports in SPDX or CycloneDX format.

🤖

AI-Powered Analysis

Use Retrieval-Augmented Generation (RAG) connected to ARM datasheets. Ask natural-language questions about hardware registers or function purposes.

🔐

Security Auditing

Identify vulnerabilities in IoT or industrial firmware. Ensure third-party components are trusted and detect code reuse or intellectual property violations.
